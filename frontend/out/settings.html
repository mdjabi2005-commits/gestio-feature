Comment obtenir ma clé Groq ?

Groq offre un moteur de traitement ultra-rapide. L'utilisation est gratuite jusqu'à un certain quota.

  1. 1.Rendez-vous sur la console Groq Cloud.Console Groq
  2. 2.Créez un compte ou connectez-vous.
  3. 3.Cliquez sur 'Create API Key' dans la section API Keys.
  4. 4.Copiez la clé (elle commence par 'gsk_') et collez-la ci-dessus.
